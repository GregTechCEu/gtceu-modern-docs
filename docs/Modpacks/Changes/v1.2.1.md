# Updating from `1.2.0` to `1.2.1`


## Custom Coils

The `coilMaterial` function now uses a supplier instead of taking a material directly.

```js
// Before:
.coilMaterial(GTMaterials.get('infinity'))

// After:
.coilMaterial(() => GTMaterials.get('infinity'))
```
